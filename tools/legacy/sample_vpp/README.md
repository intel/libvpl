sample_vpp is a oneVPL application that performs video processing of raw video sequences.  
Command Line Format:
```
sample_vpp [Options] -i InputFile -o OutputFile  
```
Sample Command Line: 
```
sample_vpp -sw 320 -sh 240  -dw 320 -dh 240 -i out.raw -o crop.raw   
```
